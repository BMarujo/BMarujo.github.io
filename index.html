Introduction

A full-stack 5G monitoring platform with real-time testing, visualization, and remote control capabilities for Smart Cities.

Android Development

Our Android app is designed to collect critical 5G metrics directly from the user's device, including signal strength, connection type, and advanced New Radio (NR) measurements. It can also run speed benchmarks using iPerf and Speedtest. These tests can be launched manually or remotely from the web platform. When a test request is sent from the web dashboard, the app runs it automatically and sends back the results in real time. All measurements are transmitted using MQTT and Kafka to the Aveiro Tech City Living Lab (ATCLL) platform, where they are processed and visualized.

Web Development

The web application presents a clear and interactive dashboard to monitor network performance across the ATCLL 5G infrastructure. Built with React and Next.js, it displays real-time and historical metrics such as signal quality, throughput, latency, and device location. Users can view network activity on a city-wide map, filter by device, gNodeB and time range, and remotely trigger speed and iperf tests on connected Android devices. This tool helps researchers and operators better understand network behavior and make smarter infrastructure decisions.
