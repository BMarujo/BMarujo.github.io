5G Smart City Data Visualization

Monitoring real-time metrics and historical data from 5G networks in a Smart City environment.

Project Overview

This project focuses on the visualization and analysis of real-time and historical data from a 5G network deployed in a Smart City. The network supports services like autonomous vehicles, drones, and sensor data collection. The aim is to create a web application for visualizing these metrics on a spatial map of the city, enabling enhanced monitoring of both network and user performance.

Key Features

  • Real-time monitoring of 5G network components like gNodeBs and UEs.
  • Visualization of network metrics such as bitrate, signal quality (SNR, CQI), and packet transmission.
  • Estimation of user location through triangulation of signal data from multiple antennas.
  • Integration with the Aveiro Tech City Living Lab platform for a comprehensive view of the Smart City's infrastructure.

Technologies Used

This project utilizes the following technologies:

  • Programming Languages: JavaScript (for frontend), Python (for backend).
  • Frontend Framework: React.js for creating dynamic user interfaces.
  • Visualization Tools: Grafana for real-time data visualization.
  • Android App: To collect physical metrics like bandwidth, RSSI, and run performance tests (e.g., iPerf).
